Master's Program · Research & Implementation

Encryption Implementation

Python Ruby AES-256 Cryptography

Wrote a research paper on Modern Cryptographic Concepts and implemented round one of AES-256 in Python, following NIST FIPS 197 standards. The implementation covers the Substitution Box for confusion, Shift Rows and Mix Columns for diffusion, and Key Addition — all applied repeatedly across rounds.

Also includes a Ruby program for password guessing. The paper provides a detailed walk-through of the code, covering the S-Box, Galois Field multiplication in MixColumns, and the RCON variable used for key expansion.
